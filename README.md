## Install process

### Clone repo

```
$ git clone https://github.com/JimmyCamus/Portfolio.git
```

### Move to project folder

```
$ cd Portfolio
```

### Install node module

```
$ pnpm install
```

### Env config

You can view the ".env.example" file to see which env variables the project needs.

### Start developer server

```
$ pnpm run dev
```
